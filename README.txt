This project is an airport simulation that attempts to determine what operations of an airport
take the longest.  The simulation has numerous variables that are outlined below and have been
run numerous times to get averages for each value.  I measure the total time taken for an empty airport
with one runway to process 1000 airplanes.  The process is as follows:
1. The aircraft lands (this is a set amount of time that blocks the runway usage)
2. The aircraft is inspected and refueled (set amount of time to wait)
3. The aircraft begins loading luggage (luggage is loaded onto a truck with a weight limit of how much it can hold, then the truck 
must drive the luggage to the plane, drop it off, then return)
4. The passengers must load on (Each passenger has a time value for how long it takes to be seated)
5. The aircraft requests access to the runway
6. When the aircraft gets the runway, it takes off (set amount of time that is blocks runway usage)

While this simulation is not very accurate, it served as a great learning experience.
I believe that this simulation does not have enough detail to accurately simulate an airport,
but in order to do it accurately there are many details that were out of the scope of this project.
If I could go back, I would have chosen a more simple system to simulate with fewer variables.

To execute, you need to have the SimPy package. Was tested on my Windows machine using Python 3.8.6.

To get SimPy, run:  
    pip install simpy

Began project from my assignment 2, then modified to meet my needs.  Primarily used the SimPy documentation.


There are two files
    The first one is finalProject.py
    This contains the simulations for 10 airplanes.  It outputs often and gives you info on the state of the system

    The second one is finalProjectAverages.py
    This contains simulations for 1000 airplanes.  It only outputs at the very end and gives you the total time for each simulation

To run the first file:

    1. It is best to clear your terminal window
    2. Execute the command: python finalProject.py

To run the second file:

    1. It is best to clear your terminal window
    2. Execute the command: python finalProjectAverages.py

NOTE: It takes me about 10 minutes to run the second file on my desktop


Values for first simulation:
    aircraftInterval = 100 #how often new aircraft arrive
    aircraftSize = 100 #how many passengers on aircraft

    landingTime = 300 #how long it takes for aircraft to land
    takeoffTime = 180 #how long it takes for aircraft to takeoff
    inspectionAndRefuelTime = 1200  #how long it takes for aircraft to be inspected and refueled
    passengerCheckInTime = 15 #How long it takes for a passenger to check in

    maxDistance = 10 #maximum luggage distance (time taken to load)
    minDistance = 5 #minimum luggage distance (time taken to load)
    maxWeight = 50 #maximum luggage weight
    minWeight = 25 #minimum luggage weight
    maxSpeed = 20 #maximum passenger speed (time taken to seat)
    minSpeed = 120 #minimum passenger speed (time taken to seat)

    truckTime = random.uniform(120, 180) #time it takes for truck to deliver luggage
    maxTruckWeight = 40000 #weight the truck can hold

    randomizePassengers = True #randomize passenger seating order
    loadSeparately = False #run the inspection, luggage, and passenger loading separately
    priorityRunway = -1 #-1 for no priorities, 0 for landing priority, 2 for takeoff priority
